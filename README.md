# 📚 Library Management System (Python)

A simple and beginner-friendly **Command Line Interface (CLI)** based Library Management System built using Python.
This project helps manage books in a library by allowing users to add, view, issue, and return books.

---

## 🚀 Features

* ➕ Add new books
* 📖 View all available books
* 📕 Issue books
* 🔄 Return books
* ❌ Exit program safely

---

## 🛠️ Technologies Used

* Python (Core concepts)
* CLI (Command Line Interface)

---

---

## ▶️ How to Run the Project

1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open terminal/command prompt
4. Navigate to project folder
5. Run the following command:

```
python main.py
```
